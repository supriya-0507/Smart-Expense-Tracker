Smart Expense Tracker

A Python-based Smart Expense Tracker application that allows users to record daily expenses, store them persistently using CSV files, and view expense history through a menu-driven interface.

 Features

Add daily expenses with date, category, and amount

Store expense data in CSV format

View complete expense history

Simple and beginner-friendly console interface

 Technologies Used

Python

CSV file handling

VS Code

GitHub

 Learning Outcomes

Practical understanding of Python file handling

Experience with real-world data storage

Familiarity with version control using GitHub
